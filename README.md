## Installation

Install individual plugins by path:

```bash
# Install a single plugin
claude /plugin install /path/to/skill-library/productivity

# Use --add-dir for development
claude --add-dir /path/to/skill-library/engineering

# Load a combination of plugins during development
claude \
  --add-dir /path/to/skill-library/product-management \
  --add-dir /path/to/skill-library/design \
  --add-dir /path/to/skill-library/engineering
```

## Adding a New Plugin

1. Create a new directory at the repo root: `my-category/`
2. Add `.claude-plugin/plugin.json` with name, description, version
3. Add skills under `my-category/skills/<skill-name>/SKILL.md`
4. Update this README

## SKILL.md Format

```yaml
---
name: skill-name
description: >
  What this skill does.
  TRIGGER when: user says "keyword1", "keyword2", or asks about topic.
argument-hint: "[arg1] [arg2]"
---

# Skill Title

Instructions, templates, and quality standards...
```
