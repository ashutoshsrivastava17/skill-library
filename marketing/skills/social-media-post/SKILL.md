# Social Media Post

Create platform-optimized social media content with engagement hooks, hashtag strategy, visual direction, and scheduling recommendations. Supports single posts, threads, carousels, and cross-platform adaptations.

---

## Process

### Step 1 — Clarify Platform, Goal, and Context

Gather the inputs needed to write effective platform-native content.

| Input | Required? | Notes |
|---|---|---|
| Platform | Yes | LinkedIn, Twitter/X, Instagram, Facebook, TikTok, Threads, YouTube Community |
| Topic / message | Yes | What is this post about? |
| Goal | Yes | Awareness, engagement, traffic, leads, community |
| Audience | Recommended | Who follows this account? B2B, B2C, industry? |
| Brand voice | Recommended | Formal, casual, witty, authoritative? |
| Visual assets | Optional | Do they have images, video, or need direction? |
| Link to include | Optional | URL for traffic-driving posts |

### Step 2 — Apply Platform-Specific Formatting

Each platform has its own constraints, norms, and best practices.

| Platform | Character Limit | Best Formats | Optimal Length | Link Behavior |
|---|---|---|---|---|
| **LinkedIn** | 3,000 chars | Text, carousel, document, video | 800-1,300 chars for engagement | Links in comments perform better |
| **Twitter/X** | 280 chars (free) / 25K (premium) | Text, image, thread, poll | 70-100 chars for retweets, 200-280 for engagement | Links reduce reach; use sparingly |
| **Instagram** (feed) | 2,200 chars | Image, carousel, Reel | 125-150 chars for casual, up to 2K for storytelling | No clickable links in captions |
| **Instagram** (Stories) | Text overlay | Image/video + stickers, polls | Minimal text, visual-first | Link sticker available |
| **Facebook** | 63,206 chars | Image, video, link post | 40-80 chars for engagement | Native video outperforms links |
| **TikTok** | 2,200 chars | Video (15s-10min) | 21-34 seconds for best completion rate | Link in bio only (unless shop) |
| **Threads** | 500 chars | Text, image | 100-300 chars | Links allowed, carousel supported |
| **YouTube Community** | 5,000 chars | Text, image, poll | 200-500 chars | Links allowed |

### Step 3 — Write the Hook

The first line determines whether someone stops scrolling. It must appear before the "...see more" truncation.

| Hook Type | Example | Best For |
|---|---|---|
| **Bold claim** | "Most marketing advice is wrong. Here's why." | Thought leadership, contrarian takes |
| **Surprising stat** | "93% of B2B buyers start with a search engine." | Data-driven content |
| **Question** | "What would you do with 10 extra hours per week?" | Engagement, community building |
| **Story opener** | "Last month I got fired. Best thing that happened." | Personal brand, narrative posts |
| **How-to promise** | "How to write a landing page in 30 minutes (framework)" | Educational content |
| **List preview** | "5 tools I use every day that cost $0:" | Utility, save-worthy content |
| **Relatable pain** | "Staring at a blank content calendar again?" | Problem-aware audiences |
| **Contrarian** | "Stop A/B testing your emails. Do this instead." | Engagement bait, strong opinions |

Truncation points (characters visible before "...see more"):
- LinkedIn: ~210 characters on mobile
- Instagram: ~125 characters
- Facebook: ~80 characters on mobile feed
- Twitter/X: Full tweet visible (no truncation at 280)

### Step 4 — Write the Body

Follow the appropriate structure for the platform and format.

**Text post structure** (LinkedIn, Twitter/X, Facebook):

```
[Hook — 1-2 lines]

[Space]

[Value / Story / Insight — 3-8 lines]
- Use line breaks for readability
- One idea per line
- Short sentences

[Space]

[CTA or engagement prompt]

[Hashtags (if applicable)]
```

**Carousel / Document post structure** (LinkedIn, Instagram):

| Slide | Content | Purpose |
|---|---|---|
| 1 (Cover) | Bold headline + visual hook | Stop the scroll |
| 2 | Problem statement or context | Build relatability |
| 3-7 | One key point per slide | Deliver value |
| 8 (optional) | Summary or recap | Reinforce learning |
| 9 (Final) | CTA — follow, save, share, link in bio | Drive action |

Design guidelines for carousels:
- 1080x1350px (4:5 ratio) for maximum feed real estate
- Consistent brand colors and fonts across all slides
- Large, readable text (minimum 24pt equivalent)
- One idea per slide — do not overcrowd

**Thread structure** (Twitter/X, Threads):

| Tweet # | Content | Purpose |
|---|---|---|
| 1 | Hook + promise ("7 lessons from scaling to $10M ARR. A thread:") | Earn the click to expand |
| 2-N | One point per tweet, numbered | Deliver value sequentially |
| Final | Summary + CTA ("If this was useful, repost tweet #1 so others see it") | Drive amplification |

### Step 5 — Add Hashtag Strategy

| Platform | Hashtag Best Practices | Recommended Count |
|---|---|---|
| **LinkedIn** | Use 3-5 relevant hashtags, mix of broad and niche | 3-5 |
| **Instagram** | Mix of large (1M+), medium (100K-1M), and small (< 100K) hashtags | 5-15 (down from the old 30 max strategy) |
| **Twitter/X** | 1-2 hashtags max, more reduces engagement | 1-2 |
| **TikTok** | Trending + niche hashtags in caption | 3-5 |
| **Facebook** | Hashtags have minimal impact | 0-2 |
| **Threads** | Still emerging, use sparingly | 1-3 |

Hashtag selection framework:
1. **Branded**: Your company/campaign hashtag (e.g., #YourBrandName)
2. **Industry**: Broad category (e.g., #DigitalMarketing, #SaaS)
3. **Niche**: Specific community (e.g., #ContentOps, #B2BSEO)
4. **Trending**: Currently popular and relevant (check platform trends)
5. **Descriptive**: What the post is about (e.g., #MarketingTips, #CaseStudy)

### Step 6 — Provide Visual Direction

If the user does not have a ready-made visual, provide direction for creating one.

| Visual Type | When to Use | Specs |
|---|---|---|
| Static image | Most posts | 1080x1080 (square), 1080x1350 (portrait), 1200x628 (landscape for links) |
| Carousel | Educational, list, or storytelling | 1080x1350 per slide, 2-10 slides |
| Short video | Demos, behind-the-scenes, tutorials | 9:16 for Reels/TikTok, 1:1 or 16:9 for feed |
| GIF / animation | Reactions, product demos | Keep under 15 seconds, loop cleanly |
| Screenshot | Social proof, data, tools | Annotate with highlights or callouts |
| Text-on-brand-background | Quote, stat, or announcement | Use brand colors and fonts |
| No visual (text-only) | Twitter/X text tweets, some LinkedIn | N/A |

Provide a brief for each visual:
- Subject / scene description
- Text overlay (if any)
- Brand elements to include (logo, colors, fonts)
- Mood / style (e.g., clean and modern, bold and playful)

### Step 7 — Recommend Posting Schedule

Optimal posting times (general guidelines — actual best times vary by audience):

| Platform | Best Days | Best Times (recipient's local time) | Worst Times |
|---|---|---|---|
| **LinkedIn** | Tue, Wed, Thu | 8-10am, 12pm | Weekends, after 6pm |
| **Twitter/X** | Mon-Fri | 9-11am, 1-3pm | Late night |
| **Instagram** | Mon, Wed, Fri | 11am-1pm, 7-9pm | 3-5am |
| **Facebook** | Wed, Thu, Fri | 9am-12pm | Late night, early morning |
| **TikTok** | Tue, Thu, Fri | 7-9am, 12-3pm, 7-11pm | Varies widely |

Always recommend the user check their own analytics for audience-specific insights.

## Quality Checklist

- [ ] Hook appears before the platform's truncation point
- [ ] Post length is appropriate for the platform
- [ ] Formatting uses line breaks and whitespace for readability
- [ ] Hashtags follow platform-specific best practices (count and relevance)
- [ ] CTA is clear and appropriate for the goal
- [ ] Visual direction includes dimensions and a clear brief
- [ ] Tone matches the stated brand voice
- [ ] No links in Instagram captions (not clickable)
- [ ] Thread tweets can each stand alone if viewed individually
- [ ] Cross-platform adaptation is not just a copy-paste — it is reformatted for each platform
- [ ] Post does not read like an ad unless it is explicitly a promoted post
- [ ] Accessibility: image alt text described, captions for video noted

## Edge Cases

- **Cross-platform request** ("post this everywhere"): Do NOT copy-paste the same text. Rewrite for each platform's format, norms, and audience. Provide platform-specific versions.
- **Controversial or sensitive topic**: Flag the risk. Recommend internal review before posting. Suggest neutral framing and disabling comments if needed.
- **Product launch with embargo**: Note the scheduled publish time clearly. Recommend drafting and scheduling in the platform's native scheduler or a tool like Buffer/Hootsuite.
- **No visual assets and no design capability**: Recommend free tools (Canva, Figma) with specific template suggestions. Alternatively, suggest text-only formats that perform well on the platform.
- **Personal brand vs. company brand**: Adjust tone accordingly. Personal brand posts are more narrative and first-person. Company posts are more polished and "we"-oriented.
- **Newsjacking / trending topic**: Speed matters — recommend publishing within 2-4 hours of the trend. Keep the post shorter and more reactive. Tie it clearly to the brand's expertise.
- **Negative comments expected**: Include a comment response framework — acknowledge, redirect, and escalate guidelines.
- **Long-form content to summarize**: Extract the 3-5 most shareable insights. Each insight should work as a standalone post. Link to the full piece in bio or comments.
- **User wants engagement but has a small following** (< 1,000): Recommend engagement-first tactics — comment on others' posts, join conversations, use niche hashtags, and tag relevant people (with permission). Organic reach is built through participation, not just posting.
