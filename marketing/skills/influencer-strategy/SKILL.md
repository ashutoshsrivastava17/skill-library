# Influencer Strategy
## Process
### Step 1: Define Objectives
| Goal | Metric | Target |
|------|--------|--------|
| Brand awareness | Reach, impressions | X |
| Engagement | Likes, comments, shares | X% rate |
| Conversions | Click-throughs, purchases | X |
| Content creation | Assets produced | X pieces |
### Step 2: Identify Influencers
| Tier | Followers | Engagement | Cost | Best For |
|------|----------|-----------|------|---------|
| Nano | 1K-10K | 5-10% | $50-500 | Niche, authentic |
| Micro | 10K-100K | 3-5% | $500-5K | Targeted reach |
| Mid | 100K-500K | 2-3% | $5K-25K | Balanced scale |
| Macro | 500K-1M | 1-2% | $25K-100K | Broad awareness |
| Mega | 1M+ | <1% | $100K+ | Mass reach |
### Step 3: Vet Candidates
| Criteria | Check |
|----------|-------|
| Audience alignment | Demographics match target market |
| Content quality | Consistent, on-brand aesthetic |
| Engagement authenticity | Real comments, not bot-like patterns |
| Brand safety | No controversial content/associations |
| Past partnerships | Competitor exclusivity, performance history |
### Step 4: Create Campaign Brief
| Section | Content |
|---------|---------|
| Objectives | Campaign goals and KPIs |
| Deliverables | Post types, quantities, platforms |
| Key messages | Brand talking points, required mentions |
| Creative direction | Mood, tone, do's and don'ts |
| Timeline | Content deadlines, review process |
| Compensation | Fee, gifting, affiliate codes |
| Legal | FTC disclosure, usage rights, exclusivity |
### Step 5: Measure Performance
| Metric | Formula |
|--------|---------|
| CPM | Cost / (Impressions / 1000) |
| CPE | Cost / Total engagements |
| CPC | Cost / Link clicks |
| CPA | Cost / Conversions |
| EMV | Earned media value of organic reach |
## Output Format
```markdown
## Influencer Campaign: [Name]
### Objectives: [Goals + KPIs]
### Roster: [Selected influencers with rationale]
### Budget: [Allocation by influencer/platform]
### Timeline: [Campaign calendar]
### Measurement: [Metrics and reporting plan]
```
## Quality Checklist
- [ ] Influencers vetted for authenticity and brand safety
- [ ] Campaign brief is clear and complete
- [ ] FTC/regulatory disclosure requirements addressed
- [ ] Content approval process defined
- [ ] Performance metrics tied to business objectives
## Edge Cases
- For B2B, focus on industry thought leaders and LinkedIn
- For regulated industries, legal review all influencer content
- If budget is small, prioritize nano/micro influencers for better ROI
